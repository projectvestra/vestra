# Smart Outfit Planning & Personalized Styling App {vestra}

This is a university-level mobile application built using **React Native (Expo)** as part of a team project.

## Project Overview
The app helps users plan outfits and receive personalized styling suggestions based on their preferences, body type, and constraints.

## Current Status
**Frontend development in progress**

Implemented so far:
1 Authentication UI (Login & Signup)
2 App flow control using Expo Router
3 Multi-step onboarding flow (4 steps)
4 Polished onboarding UI
5 Dummy/mock data only (no backend yet)

## Tech Stack
1 React Native
2 Expo
3 Expo Router
4 JavaScript
